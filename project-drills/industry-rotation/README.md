# Industry Rotation Strategy Backtest

Status: private reconstruction workspace.

## Question

Can industry-level prosperity and valuation signals support allocation decisions?

## Planned Methods

- clean industry-level financial and market data
- construct rolling prosperity and valuation indicators
- classify industries into signal groups
- build a simple rebalancing rule
- backtest against a benchmark
- evaluate return, volatility, Sharpe ratio, maximum drawdown, and robustness
