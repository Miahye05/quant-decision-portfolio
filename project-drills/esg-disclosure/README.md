# ESG Disclosure Data Engineering

Status: private reconstruction workspace.

## Question

How can corporate sustainability disclosures become structured and comparable data?

## Planned Methods

- define indicator dictionary
- create sample or anonymized disclosure dataset
- standardize quantitative and qualitative fields
- calculate disclosure completeness
- analyze missingness by firm and industry segment
- visualize disclosure patterns

## Skills Practiced

- data cleaning
- indicator design
- missingness analysis
- data quality checks
- visualization

## Public Release Criteria

- no internal ESG workbooks or unpublished team materials
- use synthetic, anonymized, or public sample data
- document the indicator schema and cleaning assumptions
- include reproducible completeness analysis
