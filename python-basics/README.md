# Python Basics

Purpose: rebuild the Python data analysis toolkit through small, explainable drills.

## Modules

1. Data inspection
   - read CSV / Excel files
   - inspect columns, types, missing values, duplicates

2. Data cleaning
   - standardize labels and dates
   - handle missing values
   - merge datasets

3. Aggregation and reshaping
   - groupby
   - pivot tables
   - wide-to-long and long-to-wide transformations

4. Visualization
   - line charts
   - bar charts
   - heatmaps
   - distribution plots

5. Modeling basics
   - simple regression
   - fixed effects intuition
   - result interpretation

6. Backtesting basics
   - returns
   - cumulative returns
   - volatility
   - Sharpe ratio
   - maximum drawdown

## Rule

Every notebook should include short markdown notes explaining what each step does and why it matters.
