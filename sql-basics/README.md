# SQL Basics

Purpose: rebuild SQL for business analytics, product analytics, and decision support.

## Core Query Patterns

1. Filtering and aggregation
   - `select`
   - `where`
   - `group by`
   - `having`

2. Joins
   - inner join
   - left join
   - anti-join logic
   - many-to-one joins

3. Conditional logic
   - `case when`
   - business flags
   - cohort labels

4. Window functions
   - `row_number`
   - `rank`
   - `lag`
   - rolling or cumulative calculations

5. Business analytics patterns
   - funnel conversion
   - retention cohorts
   - user segmentation
   - campaign performance

## Rule

Each SQL drill should answer a business question, not just demonstrate syntax.
